# 🚀 Neon Ascend

*(🇩🇪 Deutsche Version unten / German version below)*

A hyper-casual one-tap platformer for mobile, inspired by Geometry Dash – but vertical! This project was created as a personal **12-hour coding challenge** (split into three 4-hour sessions) to practice focus, rapid prototyping, and game mechanics in Unity.

## 🇬🇧 🎮 Gameplay & Features
* **Vertical Scrolling:** The neon cube automatically slides up the walls of an endless shaft.
* **One-Tap Controls:** A simple screen tap makes the player jump in an arc to the opposite wall.
* **Procedural Level Design:** Obstacles (neon spikes) are generated endlessly and randomly while playing.
* **Insta-Respawn:** No load times. Touching a spike destroys the player, and the level restarts instantly – creating the perfect "just one more try" loop.
* **Orb Abilities:** Floating orbs between the walls grant special abilities when tapped mid-air, like an extremely fast vertical dash.

## 🇬🇧 ⏱️ The 12-Hour Challenge (Dev Log)
The game was developed under a strict time limit with the following roadmap:
* **Session 1 (4h) - Core Mechanics:** Setup of Unity 2D physics, constant upward movement, and implementation of the wall jump.
* **Session 2 (4h) - Game Loop:** Programming the endless spawner for obstacles and the rapid game-over/restart system.
* **Session 3 (4h) - Abilities & Polish:** Integration of interactive orbs, particle effects (trail renderer & death explosion), UI, and mobile build.
